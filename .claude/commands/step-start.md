Before starting any new step:
1. Read CLAUDE.md to understand current project state
2. Read the full file structure to understand what exists
3. Run `npm run build` to make sure everything is clean
4. Read docs/BUILD_PLAYBOOK.md and find the section for the current step number from CLAUDE.md. Use the prompt, success criteria, and validation tests from the playbook as your instructions.
5. Then ask me: "Ready to start Step [N]. What should I build?"
